🎯 一、项目背景与目标

本项目旨在利用 Python (Pandas/Matplotlib) 对某电商店铺的商品销售数据进行深度分析，旨在识别销售趋势、评估商品贡献度，并为店铺的运营及经营策略调整提供科学依据。

分析目标：
1.  分析商品总销售额的月度变化趋势。
2.  分析不同商品小类销售额的月度变化趋势。
3.  （加分项）分析不同商品小类的总销售贡献度。

📊 二、数据与分析方法

| 内容 | 详情 |
| :--- | :--- |
|数据来源| 商品销售数据.xlsx（包含“信息表”和“销售数据表”两个工作表） |
|数据清洗| 1. 合并信息表和销售表（基于商品编号）。 2. 计算新特征：销售金额（销售价 * 订单数量）。 3. 提取特征：月份（从订单日期提取）。 |
|分析工具| Python 3.x, Jupyter Notebook, Pandas (数据处理), Matplotlib (可视化) |
|分析方法| 时间序列分析（月度趋势）、分组聚合（GroupBy）、折线图对比、柱状图排名。 |
